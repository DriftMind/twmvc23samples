# twMVC #23 Azure Functions 範例程式

## 活動

twMVC #23 於 2016-7-24 舉辦, 詳情可參考[活動資訊與報名](https://mvc.tw/event/2016/7/24)網頁。

## 原始碼使用說明

此處原始碼皆以 C# Script 開發。

Fork 這個專案，在新的 Function App 中，加入這個專案為 CI 來源。

## 範例目錄

- PeriodTimer  
  定時觸發器

- MakeThumb  
  Blob 觸發器

- SlashTime  
  HTTP 觸發器

- SlashCompany  
  HTTP 觸發器

- SlashMoedict  
  HTTP 觸發器
